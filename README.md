🎓 Student Tracker

A modern, feature-rich Student Management System built using Python. This application helps manage student records efficiently with a clean UI and smooth user experience.

🚀 Features
➕ Add new student records
✏️ Update existing student details
❌ Delete student records
🔍 Search students instantly
📋 Display all students in a structured table
💾 Data stored using SQLite database
🎨 Clean and modern GUI design
🛠️ Tech Stack
Language: Python
GUI: Tkinter
Database: SQLite3
📂 Project Structure
student-tracker/
│── main.py          # Main application file
│── database.py      # Database handling (CRUD operations)
│── README.md        # Project documentation
⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/your-username/student-tracker.git
Navigate to the project folder:
cd student-tracker
Run the application:
python main.py
